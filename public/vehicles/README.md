# Vehicle Images Setup

Download images from your Google Drive and place them in these folders.

## Folder Structure

Each vehicle needs a `main.jpg` (primary image) and optionally `1.jpg`, `2.jpg`, etc. for gallery views.

## Image Requirements

- **Format**: JPG or PNG (rename to .jpg)
- **Size**: Recommended 1200x800px or similar 3:2 ratio
- **Naming**:
  - `main.jpg` = Primary image (shown on cards)
  - `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg` = Gallery images

## Quick Setup

1. Download all images from Google Drive
2. Create folders for each car (already created)
3. Rename the best photo of each car to `main.jpg`
4. Place additional photos as `1.jpg`, `2.jpg`, etc.
5. Refresh the website to see changes
